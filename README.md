# UD3. JavaScript

Repositorio con las prácticas de la UD3 de JavaScript.

## Contenido

- `UD3. Practica 4`: ejercicios de manipulación del DOM y eventos.
- `UD3. Practica 5`: ejercicios de interacción con APIs y generación de código QR.
- `UD3. Practica 6`: app del tiempo con búsqueda de ciudad y visualización de datos meteorológicos.
